## Descrição

O sistema tem como objetivo realizar o controle de estoque de um almoxarifado, permitindo o gerenciamento de produtos, fornecedores, lotes e movimentações de entrada e saída. Além disso, o sistema possibilita a geração de relatórios para acompanhamento das operações realizadas e maior rastreabilidade dos itens armazenados.

Apesar do repositório constar com a participação de apenas um dos membros do grupo, todos os demais membros providenciaram os arquivos com os métodos e atributos corretos
definidos anteriormente no planejamento do projeto.
